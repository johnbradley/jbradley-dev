Welcome to my new blog! It's a static site built with [Astro](https://astro.build), written in
markdown, and hosted on Cloudflare.

Posts are plain markdown files, so writing one is as simple as:

```markdown
---
title: 'My next post'
description: 'What this post is about.'
pubDate: 2026-07-20
---

Post content goes here.
```

Code blocks get syntax highlighting out of the box, in both light and dark mode:

```js
function greet(name) {
	return `Hello, ${name}!`;
}

console.log(greet('world'));
```
